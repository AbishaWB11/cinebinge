# CineBinge

A movie recommendation app that helps users quickly decide what to watch based on mood, time, and genre.

## Features
- Pick For Me decision engine
- Dynamic movie data (TMDB API)
- Watchlist sidebar (add/remove)
- Smooth UI with React + Tailwind

## Tech Stack
React, Tailwind CSS, TMDB API

***It currently uses rule-based filtering, but I designed it in a way that it can be extended with AI recommendations later.***:)
